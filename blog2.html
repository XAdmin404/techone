How to stay motivated when learning programming?


As in any other field, and in programming, at some point you stagnate. That's why you need to stay motivated. For this, you must have a goal set when you get to work. Look, for example, my goal is to create complex web applications that make users' lives better, to make it easier. I always tell myself that I will be able to create something that will change the world, and that gives me a boost when I learn a new programming language.



Again, passion is very important when we want to learn programming. Programming is a field in which we cannot learn mechanically, so we must have a way of thinking logically, in order to be able to think programs, and without passion, this is almost impossible, because you have to dedicate yourself maximum, to have a goal. well defined before you start programming, to know why you write the lines of code. Passion and purpose are crucial when we want to learn programming, because without them we will not be able to cope along this path.
